\easysubproblem{Prove the following theorem: $\binom{n}{k}= \frac{P_{n,k}}{k!}$}\\
By definition, a permutation ($P_{n,k}$) is the number of ways to choose $k$ elements from a set of $n$ distinct elements and arrange them in a specific order.To find the number of combinations ($\binom{n}{k}$), we want to choose $k$ elements but we do not care about the order. Since any group of $k$ chosen elements can be arranged in $k!$ different ways, the permutation formula counts every unique combination $k!$ times.Therefore, to find the number of ways to simply select the group without regard to order, we must take the total number of permutations and divide out the number of internal arrangements. This gives us the formula:$$\binom{n}{k} = \frac{P_{n,k}}{k!}$$

\easysubproblem{In the above question, $\binom{n}{k}$ is expressed in terms of permutations and factorials. Re-express $\binom{n}{k}$ by \textbf{only} using factorials. Hint: substitute in the formula for $P_{n,k}$. We also did this in class. }
\\The formula for permutations of $n$ items taken $k$ at a time is:$$P_{n,k} = \frac{n!}{(n-k)!}$$Substituting this into the theorem from part (a), we get:$$\binom{n}{k} = \frac{\frac{n!}{(n-k)!}}{k!}$$Simplifying this expression yields the standard factorial formula for combinations:$$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

\easysubproblem{Prove that $\binom{n}{k} = \binom{n}{n-k}$.}\\
The formula for $\binom{n}{k}$ is:$$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$Now, let's write out the formula for $\binom{n}{n-k}$ by substituting $(n-k)$ wherever there is a $k$:$$\binom{n}{n-k} = \frac{n!}{(n-k)!(n-(n-k))!}$$Simplify the expression inside the second parenthesis in the denominator:$n - (n - k) = k$Substitute that back into the formula:$$\binom{n}{n-k} = \frac{n!}{(n-k)!k!}$$Because multiplication is commutative (the order doesn't matter, so $x \times y = y \times x$), the denominator $(n-k)!k!$ is exactly the same as $k!(n-k)!$. Therefore:$$\binom{n}{k} = \binom{n}{n-k}$$
